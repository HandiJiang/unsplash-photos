# Housing Data Explorer

This repository contains a Streamlit app and analysis for the provided `housing.csv` dataset.

## Contents
- `app.py` - Streamlit app (single-file)
- `analysis_report.md` - Project report with reasons and results
- `plots/` - Generated plots
- `requirements.txt` - Python dependencies

## How to run locally
1. Create a virtual environment and install dependencies:
```
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install -r requirements.txt
```
2. Run the app:
```
streamlit run app.py
```

## Streamlit Cloud
Deploy `app.py` to Streamlit Cloud by pushing this repo to GitHub and selecting the repo in Streamlit Cloud.
